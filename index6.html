Music Graffiti

This is a performance project of live and spontaneous music visualization focusing on the performer himself/herself. It contextualizes body movements, recording and turning them into graphics and music.

The materials used are editer p5.js using JavaScript, poseNet Model, music web software MIDI and music editer Ableton.

I first created geometric shapes and added Web MIDI using p5.js. Then I created short music clips on Ableton, which can be chosen by user to compose. At last, I practiced for rehearsed my performance for the final show.


The music visualization I create is a digital interface that draws graphics and makes music based on body movements. It first detects the position of arms, wrists, shoulders, then by controling the positions of body parts, user can draw various shapes.
